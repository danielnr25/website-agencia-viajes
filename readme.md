## App website desarrollado en REACT 

- Desarrollado en React es un clone de una agencia de viaje 
- Frontend Developer

## Deployment

Clonar repositorio

```bash
  git clone https://github.com/danielnr25/website-agencia-viajes.git
```
Instalar dependencias
```bash
  npm install 
```
Arrancar servidor 
```bash
  npm run dev 
```
